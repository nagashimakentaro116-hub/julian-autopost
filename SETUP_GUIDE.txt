================================================================
Julian Autopost — セットアップガイド
================================================================

【概要】
このシステムは毎日自動でThreadsに投稿します。
posts_queue.csvに投稿文を追加すれば、順番に自動投稿されます。


----------------------------------------------------------------
STEP 1: Facebook Developerアカウント作成
----------------------------------------------------------------

1. https://developers.facebook.com/ にアクセス
2. 右上「始める」→ Facebookアカウントでログイン
3. アプリを作成 →「その他」→「消費者」を選択
4. アプリ名を入力（例: JulianAutopost）→作成


----------------------------------------------------------------
STEP 2: Threads APIの有効化
----------------------------------------------------------------

1. 作成したアプリのダッシュボードを開く
2. 左メニュー「製品を追加」→「Threads API」を追加
3. 「Threadsを使用してログイン」を設定


----------------------------------------------------------------
STEP 3: Access Token取得
----------------------------------------------------------------

1. アプリダッシュボード →「Threads API」→「設定」
2. 「ユーザートークンを生成」をクリック
3. Threadsアカウント（@thejulianmethod）でログイン認証
4. 表示されたトークンをコピー


----------------------------------------------------------------
STEP 4: User ID取得
----------------------------------------------------------------

ターミナルで以下を実行（YOUR_TOKENを実際のトークンに置き換え）:

curl "https://graph.threads.net/v1.0/me?fields=id,username&access_token=YOUR_TOKEN"

→ 表示された "id" の数字がUser ID


----------------------------------------------------------------
STEP 5: config.txtに設定を記入
----------------------------------------------------------------

config.txtを開いて以下を記入:

THREADS_ACCESS_TOKEN=取得したトークンをここに貼る
THREADS_USER_ID=取得したIDをここに貼る
POST_TIME=07:00


----------------------------------------------------------------
STEP 6: 自動投稿を起動
----------------------------------------------------------------

ターミナルで以下を実行:

cd /Users/nagashimakentaro/Desktop/Julian_Autopost
python3 autopost.py

→ 毎日07:00（日本時間）に自動投稿が実行されます


----------------------------------------------------------------
投稿文の追加方法
----------------------------------------------------------------

posts_queue.csvをExcelまたはテキストエディタで開き、
以下の形式で行を追加:

id,text,image_path,status,scheduled_time
11,"投稿文をここに書く

Link in bio.",,pending,07:00

※ statusは必ず「pending」にすること
※ 投稿済みは自動的に「posted」に変わります


----------------------------------------------------------------
トラブルシューティング
----------------------------------------------------------------

エラー: "OAuthException"
→ Access Tokenの期限切れ。STEP 3をやり直す

エラー: "Invalid user ID"
→ User IDが間違っている。STEP 4をやり直す

投稿されない
→ ターミナルが閉じていないか確認。Mac起動時に自動実行したい場合は別途設定が必要。
