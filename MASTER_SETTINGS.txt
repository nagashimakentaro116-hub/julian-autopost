9. Threads Access Token 再取得方法
----------------------------------------------------------------

トークンの有効期限が切れた場合:
1. https://developers.facebook.com/ を開く
2. アプリ「ジュリアン」→「Threads APIにアクセス」→「設定」
3. 「ユーザートークン生成ツール」でトークンを再生成
4. config.txtのTHREADS_ACCESS_TOKENを更新
5. GitHub Secretsも更新:
   cd ~/Julian_Autopost
   gh secret set THREADS_ACCESS_TOKEN --body "新しいトークン"
